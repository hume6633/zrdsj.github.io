写在前边

看起来是要谈一些技术？😆然而只是说一些屁大点儿的小儿科。。。若是想要得到完整的Hexo_NexT博客搭建讲解，请移步

过去的六七天，在每天不那么忙的时候，在github上，利用github pages和自己5个月前🙉脑抽🙉买的域名。。。在自己pc的某个盘上搭建了博客网站，使用了博客框架Hexo及其相应主题NexT。然后通过git以及网站github,完成了本地博客的网站部署。

就像现在，你看见这博客的时候，其实你是在我的博客网站zrdsj.cc上，看着我的第四篇博客😂。

用到的工具

本节最后我提到一些工具，其实说起来的话，还应该提到材料。

⬆️个🌰‘说明一下—我有食材、调料，又有火、锅、碗、筷。。。才可能做熟了能吃的东西。

这里搭建博客需要的软件，网站，环境可以说是工具了，还有些可以称之为材料的这里就不再专门说明了。

动手做过这类事情的话，是很好理解的；没做过不理解也是没有关系，因为理不理解也就仅仅说明你意识会好一些，跟你自己动手搭建博客成功与否没关系。

本节的最后来了，说工具了：

pc net web site
Node.js 可以连接Github Github账号
Git 使用node install时有反应的npm镜像 Baidu账号
GitHub Desktop* 必要时可fq 多说账号*

有这些工具就可以实现全部的从0-100%功能的博客部署，鄙人几天（也是tn的够笨）的摸索下来总结了这些。
